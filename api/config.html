config.access_minio(path, operation, data)¶

Used to read and write to minio.

Parameters

  • tenant – default value is standard

  • bucket – nrcan-btap

  • path (str) – file path where data is to be read from or written to

  • data (Union[str, DataFrame]) – for write operation, it contains the data to be written to minio

Returns

Dataframe containing the data downladed from minio is returned for read operation and for write operation , null value is returned.
